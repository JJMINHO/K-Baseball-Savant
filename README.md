# K-Baseball Savant

> 컴퓨터 비전과 야구 데이터를 활용한 KBO 투구 분석 프로젝트

## 프로젝트 소개

**K-Baseball Savant**는 KBO 리그에서 활동하고 있는 투수의 투구 영상을 기반으로 투구폼을 분석하고, 이와 관련된 여러 투구 데이터를 함께 수집하여 분석하는 프로젝트입니다.

MLB에는 Baseball Savant와 같이 투구 추적 데이터, 타구 데이터, 선수별 세부 지표를 제공하는 분석 플랫폼이 존재합니다. 반면 KBO 리그의 경우, 이러한 고급 데이터가 일반 사용자에게 충분히 공개되어 있지 않아 일반 팬이나 개인 분석가가 다양한 관점에서 경기를 분석하는 데 한계가 있습니다.

**K-Baseball Savant**는 이러한 문제의식에서 출발한 프로젝트입니다. 본 프로젝트는 컴퓨터 비전을 활용하여 KBO 또는 국내 야구 영상에서 투수의 투구폼, 릴리스 시점, 팔 각도 등의 정보를 추출하고, 이를 구질·구속 등의 투구 데이터와 함께 저장하여 분석 가능한 데이터셋을 구축하는 것을 목표로 합니다.

특히 투구 영상에서 투수의 관절 위치를 추정하고, 릴리스 시점의 팔 각도, 어깨 각도, 팔꿈치 각도 등의 자세 정보를 계산하여 투구폼을 정량적으로 분석하고자 합니다. 추후에는 구질, 구속, 투구 결과 등의 데이터와 결합하여 투구폼과 투구 성과 사이의 관계를 분석할 수 있는 데이터셋을 구축할 예정입니다.

## 프로젝트 목표

본 프로젝트의 최종 목표는 야구 투구 영상을 입력받아 투수의 투구폼을 자동으로 분석하고, 각 투구별 주요 데이터를 저장 및 시각화하는 시스템을 구현하는 것입니다.

주요 목표는 다음과 같습니다.

1. 투구 영상에서 투수의 포즈를 추정한다.
2. 투구 동작 중 릴리스 시점을 탐지한다.
3. 릴리스 시점의 팔 각도 및 주요 관절 각도를 계산한다.
4. 좌투/우투 여부에 따라 분석 기준을 보정한다.
5. 구질, 구속 등의 투구 데이터를 가져와 영상에서 추출한 데이터와 함께 저장한다.
6. 투구폼 데이터와 투구 결과 데이터 사이의 관계를 분석한다.
7. 분석 결과를 시각화하여 직관적으로 확인할 수 있도록 한다.

## 주요 기능 계획

- 투구 영상 입력 및 프레임 추출
- 투수 포즈 추정
- 릴리스 프레임 탐지
- 팔 각도, 어깨 각도, 팔꿈치 각도 계산
- 좌투/우투 기준 보정
- 구질 및 구속 데이터 수집
- 분석 결과 CSV 또는 데이터베이스 저장
- 투구별 분석 결과 시각화
- 선수별 투구폼 변화 비교

## 기대 효과

기존의 투구폼 분석은 전문 장비나 수작업 분석에 의존하는 경우가 많습니다. 본 프로젝트는 일반적인 야구 경기 영상 또는 배터리캠 영상을 활용하여 투수의 투구폼을 자동으로 분석하는 것을 목표로 합니다.

이를 통해 투수의 팔 각도, 릴리스 포인트, 투구폼의 일관성 등을 데이터 기반으로 확인할 수 있습니다. 또한 구질·구속 데이터와 결합하면 특정 투구폼 변화가 구속, 구질, 제구 또는 투구 결과에 어떤 영향을 주는지 분석할 수 있을 것으로 기대됩니다.

장기적으로는 KBO 및 국내 야구 환경에 맞춘 투구 분석 도구로 확장할 수 있으며, 아마추어 선수 분석, 선수 성장 추적, 야구 데이터 분석 학습용 프로젝트로도 활용할 수 있습니다.
